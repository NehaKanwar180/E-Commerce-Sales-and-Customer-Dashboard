# E-Commerce Sales & Customer Insights Dashboard

Power BI project analyzing Sales, Profit, Customer Segments, and Product Performance using the Superstore dataset.

## Key Insights
- Total Sales: ₹5.05M
- Total Profit: ₹392.56K
- Top Categories: Technology, Furniture, Office Supplies
- Top States by Sales: California, New York, Texas
- Customer Segments: Consumer, Home Office, Corporate

## Files
- `E Commerce sales and customer dashboard.pdf` — Dashboard export
- `Superstore_Sales_Dataset.csv` — Raw dataset
- `Cleaned data.csv` — Cleaned dataset
- `project.pbix` — Power BI file (optional)
